# Source Code for the paper "On the Interaction of Compressibility and Adversarial Robustness"

This repository contains the source code for the paper "On the Interaction of Compressibility and Adversarial Robustness", accepted to the International Conference on Learning Representations (ICLR) 2026.

Please examine `experiment.sh` for an example experiment run. Further parameters can be found in `main.py`.

This code allows you to:
- Run experiments on any torchvision dataset
- Run experiments with torchvision models
- Utilize adversarial training
- Conduct validation-based model selection

## Coverage

The code in this repository covers the main results of the paper. Some auxiliary experiments and analyses reported in the paper rely on additional scripts that are not included here. If there is a specific experiment you would like to replicate but cannot find in this repository, please reach out to Melih Barsbey at `m.barsbey@imperial.ac.uk` and the relevant code or instructions will be made available.
